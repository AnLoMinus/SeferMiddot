# 📖 ספר המידות הדיגיטלי / אינטראקטיבי (Sefer HaMiddot)

אפליקציה אינטראקטיבית ומתקדמת ללימוד, עיון וניהול מידות על פי "ספר המידות". המערכת משלבת טקסטים קלאסיים עם טכנולוגיות AI מודרניות לחוויית למידה עמוקה.

---

## ✨ תכונות עיקריות
- 📚 **דפדוף במידות**: צפייה במידות השונות עם תמיכה מלאה בטקסט מקורי וניקוד.
- 🤖 **ממשק AI מתקדם**: שיחה עם בינה מלאכותית ללימוד ושאילת שאלות על המידות.
- ➕ **הוספת מידות חדשות**: תמיכה דינמית בהוספת מידות באמצעות קבצי Markdown.
- 🗂️ **ניהול ומיון**: לוגיקה חכמה למיון מותאם אישית של המידות בתפריט הניווט.
- 👥 **שולחן עגול (Round Table)**: מערכת דיונים בזמן אמת בין משתמשים.
- 🔐 **מערכת התחברות מאובטחת**: התחברות באמצעות Google.
- ⚙️ **ממשק מנהלים (Admin)**: ניהול משתמשים, צפייה בהיסטוריית גלישה ושאלות.
