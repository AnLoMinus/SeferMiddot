# 📖 ספר המידות הדיגיטלי (Sefer HaMiddot Digital)

ברוכים הבאים אל **ספר המידות הדיגיטלי** - אפליקציה אינטראקטיבית ומתקדמת ללימוד, עיון וניהול מידות על פי "ספר המידות" של רבי נחמן מברסלב. המערכת משלבת טקסטים קלאסיים עם טכנולוגיות AI מודרניות לחוויית למידה עמוקה, נגישה וחדשנית. ספר המידות הדיגיטלי נועד להנגיש את החכמה העתיקה לדור החדש באמצעות כלים דיגיטליים מתקדמים.

---

## ✨ תכונות עיקריות של ספר המידות הדיגיטלי
- 📚 **דפדוף במידות**: צפייה במידות השונות עם תמיכה מלאה בטקסט מקורי וניקוד. [לרשימת המידות המלאה](SeferHaMiddot.json)
- 🤖 **ממשק AI מתקדם**: שיחה עם בינה מלאכותית ללימוד ושאילת שאלות על המידות.
- ➕ **הוספת מידות חדשות**: תמיכה דינמית בהוספת מידות באמצעות קבצי Markdown (לדוגמה: [מידת החסד](new_midot/chesed.md)).
- 🗂️ **ניהול ומיון**: לוגיקה חכמה למיון מותאם אישית של המידות בתפריט הניווט.
- 👥 **שולחן עגול (Round Table)**: מערכת דיונים בזמן אמת בין משתמשים.
- 🔐 **מערכת התחברות מאובטחת**: התחברות באמצעות Google.
- ⚙️ **ממשק מנהלים (Admin)**: ניהול משתמשים, צפייה בהיסטוריית גלישה ושאלות.
- 📜 **יומן אירועים**: מעקב אחר שינויים ועדכונים במערכת. [צפה ביומן האירועים](changelog.md)
